- Что выполняет команда: git add, git init, git commit, git push?
/ git add - добавляет новые/измененные файл(ы) в репозитории
/ git init - "подключает" этот репозитории, создает его
/ git commit - создает "слепок" нашего репозитория, контрольную точку
/ git push - отправляет локальный репозиторий на удаленый репозиторий

- Как можно скопировать файл с помощью командной строки?
/ копирование происходит командой copy. 
/ копируем файл с названием TEXT.txt из каталога в папку data -> copy TEXT.txt data

- Как я могу удалить все mp3 файлы из каталога с помощью командной строки?
/ проходим до нужной нам папки или запускаем командную строку в нужной нам папке и запускаме команду del с параметром *.mp3 
/ del *.mp3
/ (*) говорит о том что нужно удалить все файлы с раширением mp3
